# 不休眠猫头鹰

Apple 芯片原生的 macOS 菜单栏工具。🦉 模式阻止系统空闲休眠，屏幕仍可正常熄灭；🐦 模式恢复系统默认休眠规则。

v2 使用 macOS LaunchDaemon 辅助程序。首次批准后，日常切换无需重复输入密码；主应用失联 15 秒会自动恢复休眠。默认仅接通电源时允许合盖守夜，也可在控制页允许电池供电。低电量与危急热状态会自动恢复🐦。

## 使用

- 左键单击菜单栏鸟类图标：切换模式。
- 右键单击：打开菜单、控制页面、登录启动和退出。
- 退出应用时会自动释放休眠控制。

## 构建与测试

```sh
swift run NoSleepOwlTests
./scripts/build-app.sh
```

生成的应用位于 `dist/不休眠猫头鹰.app`。
