# No Sleep Owl v0.1.0 App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish the first public arm64 build as `v0.1.0` with a smaller, padded, rounded macOS Launchpad icon.

**Architecture:** Keep the existing owl artwork as the brand source, transform it deterministically into a 1024px rounded master icon, and derive the complete `.icns` iconset from that master. Keep release metadata in `Info.plist`, validate both metadata and icon geometry with a repository script, then rebuild the existing app and release artifacts.

**Tech Stack:** Swift Package Manager, AppKit application bundle, zsh, `sips`, `iconutil`, `PlistBuddy`, GitHub CLI.

## Global Constraints

- Public version is exactly `0.1.0`; build number is exactly `1`.
- Distribution filenames start with `NoSleepOwl-v0.1.0-arm64`.
- Main executable and helper executable remain `arm64`.
- Existing menu bar icon, sleep behavior, monitoring, settings, and localization are unchanged.
- The existing complete artwork is clipped directly to the macOS rounded shape; no extra backing tile, inset square, or secondary padding is added.
- Existing Git history is preserved.

### Task 1: Add Release Metadata and Icon Validation ✅

**Files:**
- Create: `scripts/validate-release-assets.sh`
- Modify: `Tests/NoSleepOwlCoreTests/main.swift`

**Interfaces:**
- Consumes: `Resources/Info.plist`, `Resources/AppIcon.png`, `Resources/AppIcon.icns`
- Produces: a zero-exit validation command for version `0.1.0`, build `1`, 1024px PNG, rounded transparent corners, and a readable `.icns`

- [ ] **Step 1: Write a failing release validation script**

```zsh
#!/bin/zsh
set -euo pipefail
ROOT="${0:A:h:h}"
PLIST="$ROOT/Resources/Info.plist"
[[ "$(/usr/libexec/PlistBuddy -c 'Print :CFBundleShortVersionString' "$PLIST")" == "0.1.0" ]]
[[ "$(/usr/libexec/PlistBuddy -c 'Print :CFBundleVersion' "$PLIST")" == "1" ]]
[[ "$(sips -g pixelWidth "$ROOT/Resources/AppIcon.png" | awk '/pixelWidth/{print $2}')" == "1024" ]]
[[ "$(sips -g pixelHeight "$ROOT/Resources/AppIcon.png" | awk '/pixelHeight/{print $2}')" == "1024" ]]
iconutil --convert iconset --output "${TMPDIR:-/tmp}/NoSleepOwl-validation.iconset" "$ROOT/Resources/AppIcon.icns"
```
