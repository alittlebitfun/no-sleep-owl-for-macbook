# 不休眠猫头鹰 v0.1.0 与启动台图标调整设计

## 目标

将首次公开版本统一调整为 `0.1.0`，并修复启动台图标主体过大、缺少系统化圆角和视觉重量偏高的问题。

## 图标设计

- 延续当前立体猫头鹰形象，避免更换品牌主体。
- 直接对现有完整图像应用 macOS 圆角遮罩，保留原图自身的深蓝背景和主体比例。
- 不额外添加底板、内嵌方框或二次留白，圆角比例遵循 macOS App 图标观感。
- 圆角直接绘制在图标资源中，透明画布之外不残留直角背景。
- 生成完整的 `.icns` 多尺寸资源，覆盖 16、32、128、256、512 和 1024 像素场景。
- 重点检查启动台、访达和“关于”界面的 64、128、256、512 像素显示效果。

## 版本与发布

- 应用短版本号统一设为 `0.1.0`。
- 首次构建号设为 `1`。
- 安装包命名统一为 `NoSleepOwl-v0.1.0-arm64`。
- 删除 GitHub 上现有的 `v2.0.0` Release 及远程标签。
- 创建 `v0.1.0` 标签与 Release，上传 DMG、ZIP 和 SHA256 校验文件。
- 保留 Git 提交历史，不改写已推送的代码提交。

## 实现边界

- 不修改菜单栏鸟类图标和猫头鹰/小鸟模式交互。
- 不修改防休眠、合盖运行、温度监控或高占用应用功能。
- 本次只调整应用图标资源、版本元数据、打包产物和公开 Release。

## 验证标准

- 图标源文件为 1024×1024，并能生成结构完整的 `.icns`。
- 应用元数据显示 `0.1.0 (1)`。
- Apple 芯片 release 构建成功，主程序与辅助程序架构均为 `arm64`。
- 全量自动测试通过。
- 新应用可覆盖安装并正常启动，菜单栏功能和设置页面可用。
- 启动台图标边缘为圆角，整体大小与相邻系统应用接近，没有内嵌方框。
- DMG 和 ZIP 可正常打开，SHA256 与上传文件一致。
- GitHub 只保留本次公开版本 `v0.1.0`。
