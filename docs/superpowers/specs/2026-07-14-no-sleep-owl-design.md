# 不休眠猫头鹰：产品与技术设计

## 产品目标

「不休眠猫头鹰」是一款轻量的 macOS 菜单栏应用。用户单击菜单栏鸟类图标，即可控制 Mac 是否允许系统空闲休眠。屏幕仍可按系统设置正常熄灭，下载、训练、渲染等后台任务继续运行。

应用追求三个结果：一眼看懂当前状态、一次点击完成切换、不需要管理员权限。

## 产品范围

首个版本提供：

- 🦉 猫头鹰模式：阻止系统因空闲而休眠，允许显示器熄灭。
- 🐦 小鸟模式：释放应用创建的休眠阻止，让 macOS 恢复正常休眠规则。
- 菜单栏常驻图标与左右键操作。
- 独立控制页面。
- 本次猫头鹰模式运行时长。
- 开机登录后自动启动选项。
- 退出应用前自动释放休眠阻止。

首个版本不提供定时任务、按应用触发、下载检测、历史统计和合盖防休眠。Mac 合盖后的行为继续服从系统与硬件规则。

## 交互设计

### 菜单栏

- 小鸟模式显示单色小鸟图标。
- 猫头鹰模式显示单色猫头鹰图标。
- 左键单击图标，立即切换模式。
- 右键单击图标，打开快捷菜单。

快捷菜单包含：

1. 当前状态，只读展示。
2. “切换到猫头鹰模式”或“切换到小鸟模式”。
3. “打开不休眠猫头鹰”。
4. “登录时自动启动”开关。
5. “退出不休眠猫头鹰”。

状态切换成功后，菜单栏图标、菜单文字、控制页面同步更新。切换失败时保留原状态，并在控制页面显示简短错误提示。

### 控制页面

控制页面使用单窗口、单页面结构，包括：

- 当前鸟类的大号图标。
- “猫头鹰正在守夜”或“小鸟可以休息”的状态文案。
- 一个醒目的模式切换按钮。
- 猫头鹰模式本次持续时长；小鸟模式下显示“尚未开始守夜”。
- “登录时自动启动”开关。
- 对屏幕行为的说明：“屏幕可以熄灭，后台任务会继续运行。”

应用默认不显示 Dock 图标。用户通过右键菜单打开控制页面；关闭窗口后应用继续驻留菜单栏。

## 启动与退出规则

- 首次启动和普通重新启动均进入小鸟模式。
- 登录时自动启动只负责启动应用，不自动进入猫头鹰模式。
- 正常退出、应用终止和可处理的异常退出路径都要释放休眠阻止。
- 应用不改写 macOS 的节能、电池或锁屏设置。

这些规则优先避免用户忘记关闭后造成持续耗电。

## 技术设计

使用 Swift、SwiftUI 与 AppKit 构建原生 macOS 应用，最低系统版本根据本机可用 SDK 确定并在工程中明确记录。

核心组件：

- `SleepAssertionController`：通过 IOKit 电源管理接口创建和释放“阻止系统空闲休眠”的 assertion。它只管理后台系统休眠，不阻止显示器休眠。
- `OwlModeStore`：维护当前模式、开始时间、持续时长和错误状态，作为菜单栏与控制页面的唯一状态来源。
- `StatusItemController`：管理菜单栏状态项，区分左键与右键操作，更新图标和快捷菜单。
- `ControlWindow`：SwiftUI 控制页面，读取并操作 `OwlModeStore`。
- `LaunchAtLoginController`：使用 macOS ServiceManagement API 管理登录时自动启动。
- `AppLifecycleController`：负责启动默认状态、窗口展示与退出清理。

状态流：用户触发切换后，`OwlModeStore` 请求 `SleepAssertionController` 执行电源操作；成功后发布新状态，菜单栏和窗口随之刷新。失败时状态保持不变并发布错误信息。

## 图标设计

- 应用图标以 🦉 为主体，使用简洁、居中的 macOS 圆角图标构图。
- 菜单栏图标使用模板图像，适配浅色与深色菜单栏。
- 猫头鹰与小鸟轮廓必须有明显差异，缩小到菜单栏尺寸后仍可辨认。
- Emoji 可作为视觉原型；发布构建使用项目内置图像资源，避免不同系统字体造成外观漂移。

## 错误处理

- 创建电源 assertion 失败：保留小鸟模式，展示“未能开启守夜，请稍后重试”。
- 释放 assertion 失败：继续显示猫头鹰模式和错误提示，允许用户重试；退出流程仍再次尝试释放。
- 登录启动权限变更失败：还原开关并显示系统返回的可理解提示。
- 不记录个人数据，不访问网络，不发送遥测。

## 测试与验收

自动化测试覆盖：

- 小鸟模式成功切换到猫头鹰模式。
- 电源 assertion 创建失败时状态不变。
- 猫头鹰模式成功切换到小鸟模式。
- 释放失败时保持猫头鹰状态。
- 本次运行时长从进入猫头鹰模式开始计算，离开后清零。
- 退出清理会请求释放有效 assertion。
- 登录启动开关成功与失败时的状态同步。

人工验收：

1. 应用启动后只在菜单栏显示小鸟图标。
2. 左键单击切换到猫头鹰图标，再次单击恢复小鸟图标。
3. 猫头鹰模式下让屏幕按系统设置熄灭，Mac 后台任务继续运行。
4. 右键菜单可切换模式、打开控制页面、设置登录启动并退出。
5. 控制页面与菜单栏状态始终一致。
6. 退出后通过系统电源状态工具确认应用不再持有休眠 assertion。
7. 应用无需管理员权限即可完成全部功能。
